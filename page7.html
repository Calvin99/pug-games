PhysX Sim:

This sandbox physics game has lots of easter eggs to find, get searching!

Controls:

Mouse

About Physx Sim

    CONTROLS:
  • ESC - Hide Info
  • Left Click - PAINT
  • Right Click - ERASE
  • # 1-9 - Change Brush Thickness
  • F KEY - Cycle Paint Mode
  • L KEY - Adjust Grid
  • X KEY - Clear
  • M KEY - Toggle Gate Blocks
  • ARROWS - Move Slider Blocks
  • SPACE - Pause Sim
  • SHIFT - Linear Paint Mode
    • BONUS:
  • G KEY - Reverse Gravity

This prototype is meant for running simple physics simulations. It is not yet capable of actual science, however, development is ongoing. More features coming soon!
